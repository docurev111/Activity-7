# Task Management System - Frontend

This is the frontend UI for the Task Management System built with React and Vite.

## Technologies Used

- **React**: UI library
- **Vite**: Build tool
- **Axios**: HTTP client
- **CSS3**: Styling

## Features

- Dashboard view with statistics
- Project management (create, view, delete)
- Task management (create, view, update status, delete)
- User management (create users)
- Task filtering by project
- Progress tracking
- Deadline monitoring with overdue alerts
- Status updates (To Do, In Progress, Completed)
- Responsive design

## Installation

```bash
npm install
```

## Running the Application

```bash
# Development mode
npm run dev
```

The app will run on http://localhost:5173

## Usage

1. **Add Users**: Click "Add User" to create team members
2. **Create Projects**: Click "Add Project" to create new projects
3. **Add Tasks**: Click "Add Task" to assign tasks to projects and users
4. **View Tasks**: Click "View Tasks" on any project card to see its tasks
5. **Update Status**: Use the dropdown in each task to update its status
6. **Track Progress**: See completion progress bars on each project

## API Connection

Make sure the backend API is running on http://localhost:3000 before starting the frontend.
